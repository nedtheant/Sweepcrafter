# Sweepcrafter

This is a version of Minesweeper implemented in Minecraft (1.16.4) when I was in highschool. Thought I'd put it on github for safekeeping.

In the folder:

- `pack` we have the resource pack to show the numbers, buttons, mines, etc.
- `world` we have the world for playing the minesweeper game.

To play:

1. Click the button to start a new game, beginner, intermediate etc.
1. Right-click to flag/un-flag a spot.
1. Left-click to probe.
